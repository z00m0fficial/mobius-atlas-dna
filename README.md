# Mobius Atlas DNA

## Description
Living visualization engine for Mobius Organizational Intelligence. Atlas DNA renders verified organizational learning as evolving DNA strands, department branches, memory nodes, intelligence pulses, and industry-specific genome expressions.

## Purpose
Atlas DNA makes intelligence visible. It is not decorative animation; every visible change must correspond to a real Mobius event such as memory creation, verification, recommendation acceptance, risk detection, or organizational evolution.

## Core Responsibilities
- Render organization, department, individual, and project DNA.
- Visualize memory growth and learning velocity.
- Represent industry genome differences.
- Emit UI-ready DNA state for the Command Center.
- Maintain honest mapping between system events and visual mutations.

## Connected Systems
- mobius-memory
- mobius-genome-registry
- mobius-command-center
- mobius-pulse
- mobius-event-bus
